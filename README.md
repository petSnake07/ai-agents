# AI Agents Portfolio

A collection of Artificial Intelligence algorithms demonstrated through intelligent agents. 

## Topics Covered

### Uninformed Search

**File:** `uninformed_search.ipynb`

Implemented classical graph-search algorithms:

* Depth-First Search (DFS)
* Breadth-First Search (BFS)
* State-space exploration
* Pathfinding and search-tree analysis

Key Concepts:

* Completeness
* Optimality
* Time and space complexity

---

### Informed Search

**File:** `informed_search.ipynb`

Implemented heuristic-based search techniques:

* A* Search
* Heuristic evaluation functions
* Cost-based pathfinding

Key Concepts:

* Heuristic admissibility
* Optimal search
* Search efficiency

---

### Adversarial Search

**File:** `adversarial_search.ipynb`

Developed game-playing agents using adversarial search techniques:

* Minimax Search
* Game-tree evaluation
* Turn-based strategic decision making

Applications:

* Tic-Tac-Toe AI
* Competitive game environments

---

### Probabilistic Reasoning

**File:** `beliefs.ipynb`

Implemented intelligent agents capable of reasoning under uncertainty.

Key Concepts:

* Belief states
* Probabilistic inference
* Decision making with incomplete information

Applications:

* Hidden target localization
* Autonomous agent reasoning

---

### Constraint Satisfaction Problems (CSPs)

**File:** `queens.ipynb`

Implemented CSP-solving techniques using the N-Queens problem.

Key Concepts:

* Backtracking search
* Constraint propagation
* State-space reduction

Applications:

* Scheduling
* Resource allocation
* Puzzle solving

---

### Machine Learning Classification

**File:** `classifiers.ipynb`

Implemented machine learning classifiers using Python and NumPy.

Key Concepts:

* Supervised learning
* Feature vectors
* Classification
* Numerical optimization

Applications:

* Pattern recognition
* Predictive modeling
